Tecnología y Arquitectura Serverless

  • Frontend: SPA Reactiva y Moderna
    React 18 empaquetado con Vite. Utiliza TypeScript estricto, TailwindCSS para estilos, y componentes accesibles mediante Radix UI y shadcn/ui.
  • Backend: BaaS (Backend as a Service)
    Infraestructura alojada en Supabase (PostgreSQL). Gestiona Auth, Storage para evidencias y Edge Functions para lógica de negocio en la nube.
  • Seguridad Inquebrantable (RLS)
    Manejo de sesiones con JWT nativo. Políticas Row Level Security obligatorias: cada cliente solo puede acceder (SELECT, INSERT, UPDATE) a las filas vinculadas a su `company_id`.
